## ⭐ Highlighted projects

### 🚗 BMW Group · Critical TechWorks *(current)*

**Quality — Incident & Problem Management**
A full-stack platform (front-end heavier) for the BMW Group's vehicle-quality processes, handling incident and problem management at enterprise scale. I develop and maintain features, write unit (Jest) and end-to-end (Playwright) tests, and support CI/CD across distributed, multi-country teams.

`Angular (v19+)` · `Java / Quarkus` · `Kafka` · `Docker` · `Azure` · `GitHub`

**E-Commerce Ecosystem — BMW · MINI · Motorrad**
Feature development inside a large-scale, high-complexity e-commerce architecture, integrating multiple systems and services. Strong focus on cross-team collaboration, technical investigation and reliable delivery.

`Next.js` · `TypeScript` · `Tailwind` · `AWS (Lambda, REST APIs)` · `Jest`

### 🏥 SPMS via Capgemini — Healthcare Professional Management System

A React web app for a **national public healthcare agency**, streamlining the registration, evaluation and management of newly graduated doctors. I built interfaces and features across certificates, statements, competitions and residency/internship workflows, with API integration in a regulated public-sector environment.

`React.js` · `REST APIs` · `Docker` · `Azure DevOps` · `Postman`

### 🏭 Shop Floor Management — *Bridgestone*

Modernised production-sector management, replacing manual, paper-based processes with a **dynamic React web app** integrated with the **PI System** to collect data from databases and factory machines in **real time**. The UI is organised into four dashboards with draggable cards tailored to operational, supervisory and management levels.

**My contribution:** UI prototyping & design (Figma), building the interface and features in React, data-treatment logic, integrating multiple APIs/endpoints, and applying business rules.

`React.js` · `PI System (PI AF, PI Vision, PI Web API)` · `SQL Server` · `Microservices` · `.NET 6` · `Azure DevOps` · `Figma`

### 🏆 Top Team Members — *Bridgestone*

A web app that recognises the most productive employee each month based on KPIs, while managing machine login/logout data and tracking work hours. Translated a complex set of business rules into an efficient, user-friendly interface.

**My contribution:** UI development and design improvements in React, data-treatment logic, API integration, and implementation of intricate business rules.

`React.js` · `PI System` · `SQL Server` · `Microservices` · `.NET 6` · `Azure DevOps` · `Figma`

### 📱 BMX Tracking App — *AMBEV*

An Android app for **cloud-based recipe management**, assisting operators during production by validating recipes against target quantities. Uses **embedded Machine Learning (OCR)** to recognise label text through a tablet camera.

**My contribution:** design patterns, UX research & usability, and UI prototyping in Figma.

`Android Studio` · `Azure` · `Azure DevOps` · `SonarCloud` · `SQL Server` · `Figma` · `Miro`

### 📊 Dashboards Management — *S.Build Automatization & IT*

My first professional project as an independent front-end developer: a React web app for building and optimising dashboard interfaces, with reusable components, REST API integration and a strong UX/UI focus.

`React.js` · `JavaScript` · `HTML/CSS` · `Figma` · `Azure DevOps`
